# Lumora • Agentic Code Intelligence Platform

</summary>

### Overview

Lumora is an enterprise-grade AI platform that enables developers to understand large GitHub repositories using natural language. Instead of searching manually through thousands of files, users can ask architectural and implementation questions while an AI agent retrieves relevant code context using semantic search and repository structure analysis.

The project is designed around production software engineering principles, emphasizing scalability, modularity, and maintainability.

---

| Category | Details |
|-----------|---------|
| **Stack** | Python • FastAPI • LangGraph • Qdrant • PostgreSQL • Redis • Docker • Tree-sitter |
| **Architecture** | Agentic AI • Hybrid Retrieval • REST APIs • Modular Services |
| **Performance** | Optimized semantic retrieval with repository-aware indexing |
| **Security** | Environment-based secrets, isolated containers, API validation |
| **Scalability** | Containerized microservice-friendly backend |
| **Impact** | Enables developers to understand unfamiliar repositories significantly faster |
| **Repository** | **Coming Soon** |

### Highlights

- Built an Agentic AI workflow using **LangGraph ReAct Agents**
- Repository-aware semantic retrieval
- Tree-sitter AST parsing
- Hybrid vector + structural search
- Tool-calling architecture
- Repository analytics
- FastAPI production backend
- Dockerized deployment
- PostgreSQL + Redis + Qdrant integration
- Designed for enterprise-scale repositories

# RAG Chatbot • Enterprise Document Intelligence

</summary>

### Overview

A production-ready Retrieval-Augmented Generation chatbot capable of ingesting PDF documents, generating embeddings, retrieving relevant context, and producing source-grounded responses using Large Language Models.

The project follows a layered architecture, separating ingestion, retrieval, generation, and presentation into independent modules.

---
